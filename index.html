OBJETIVOS

  • Mostrar de forma virtual el objeto de información OVI su diseño web para creación de contenidos en formato web.
  • Dar a conocer la información respecto al diseño de sitios web para la visualización de los diferentes contenidos.
  • Mostrar el manejo y el uso de GIT y GITHUB para gestionar proyectos y diseños web.
  • Conocer los diferentes editores de código y herramientas de sitios web.

    • DEFINICION

    Un editor de código permite editar texto ylo cual sirve para editar el codigo fuente de programas informáticos. Este contiene caracteristicas que permite acelerar la escritura del codigo .mediante el resaltado de sintaxis, autocompleta y llaves. Muchos editores de codigo verifican la correcta escritura del codigo y muestran el error en caso de existir.
